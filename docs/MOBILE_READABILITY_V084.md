# v0.8.4 모바일 웹 가독성 보강

목표: PWA 기능 없이 모바일 웹만 개선한다.

## 반영 기준

- 노안 사용자를 고려해 모바일 본문·안내·체크박스·입력 라벨 글자 크기를 키운다.
- 핵심 경고와 유료 리포트 설명은 줄간격을 넓혀 읽기 쉽게 만든다.
- 버튼과 선택지 터치 영역을 크게 유지한다.
- 입력칸 placeholder와 라벨 대비를 높인다.
- 법률자문·세무자문·유언장 작성 대행으로 보일 수 있는 새 문구는 추가하지 않는다.
- PWA, 앱 설치 유도, 앱 아이콘, manifest는 추가하지 않는다.
- 한글 깨짐 방지를 위해 UTF-8 인코딩을 유지한다.
