# v0.9.1 공개 문구 최종 정리

고객 화면에서 미완성 서비스로 오해될 수 있는 표현을 추가 정리했습니다.

## 수정 내용

- 리포트 준비 알림 받기 -> 리포트 안내 받기
- 준비되는 순서대로 -> 선택하신 안내를 먼저
- 리포트가 준비되면 -> 리포트 안내를
- 리포트 출시와 서비스 안내 -> 리포트와 서비스 안내
- 리포트 생성 -> 리포트 정리

## 유지 원칙

- PWA 추가 없음
- 실제 결제 기능 추가 없음
- 법률자문, 세무자문, 유언장 작성 대행, 전문가 추천 표현 없음
- 모바일 입력폼 위치와 가독성 유지
