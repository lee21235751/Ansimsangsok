# 안심상속 플랫폼 리스크·수익성 상시 검토 체크리스트

## 1. 사업자 리스크 제거 원칙

안심상속은 법률자문, 세무자문, 유언장 작성 대행, 변호사 추천·알선 서비스를 제공하지 않는다.
안심상속의 서비스 범위는 다음으로 제한한다.

- 상속분쟁 위험 신호 확인
- 유류분·부동산·국제상속 검토 필요 신호 안내
- 법률용어를 쉽게 설명하는 정보성 리포트
- 공증 전 유언 준비자료 정리
- 상담 전 준비자료 정리
- 광고 또는 제휴 전문가 목록 제공

금지 표현:

- 법적으로 유효합니다
- 유류분 청구가 가능합니다/불가능합니다
- 상속세는 얼마입니다
- 상속분쟁을 해결합니다
- 변호사를 추천합니다
- 공증만 받으면 됩니다
- 소송을 막아드립니다

허용 표현:

- 검토 신호가 있습니다
- 확인이 필요할 수 있습니다
- 상담 전 준비자료를 정리합니다
- 전문가 목록을 확인할 수 있습니다
- 광고 또는 제휴 프로필이 포함될 수 있습니다

## 2. 변호사법 리스크 체크

- 상담 연결 1건당 수수료 금지
- 수임료 또는 성공보수 배분 금지
- 특정 변호사 자동 추천·배정 금지
- 전문가 목록은 광고/제휴 프로필임을 표시
- 고객이 직접 비교·선택한다는 문구 표시
- 상담·선임 계약은 고객과 전문가가 직접 체결

안전한 수익 방식:

- 전문가 프로필 월정액 광고비
- 분야별 카테고리 노출료
- 콘텐츠 제휴비
- 웨비나/세미나 광고비

## 3. 개인정보 리스크 체크

무료 진단 단계에서 받지 않는다:

- 주민등록번호
- 정확한 주소
- 가족관계증명서
- 등기부등본
- 계좌번호
- 상속인 전체 실명
- 상세 재산 목록

무료 진단에서 허용:

- 이름
- 연락처
- 이메일
- 대략적 자산 구간
- 가족관계 유형
- 부동산 유형과 개수
- 해외 거주·외국 국적·해외 재산 여부

필수 문구:

- 개인정보 수집·이용 동의
- 마케팅 수신 선택 동의
- 보유 기간
- 삭제 요청 가능성

## 5. 매 개발 단계 사전 확인

- 법률판단처럼 보이는 문구가 없는가?
- 변호사 추천·알선처럼 보이는 흐름이 없는가?
- 개인정보 수집이 과하지 않은가?
- 광고/제휴 표시가 필요한 곳에 들어갔는가?
- 무료 진단 뒤 유료 리포트 또는 자료정리 상품으로 자연스럽게 이어지는가?
- 일반인이 이해할 수 있는 쉬운 문구인가?
- 고객이 불필요하게 불안해하거나 반대로 결과 보장으로 오해하지 않는가?


## v0.6.6 추가 검토

### 리포트 상품 리스크
- 리포트는 법률자문/세무자문/유언장 작성 대행이 아니라 정보성 위험 신호 정리 자료로 표기한다.
- 유효/무효, 가능/불가능, 해결/보장, 추천/배정 표현을 사용하지 않는다.
- 자동 리포트는 템플릿 기반 문구로 시작하고, 고가 자료정리 상품은 별도 검수 단계로 분리한다.

### 수익 극대화
- 무료 결과 직후 리포트 미리보기를 보여 구매 이유를 만든다.
- 1차 저가 리포트 → 부동산 상세 → 공증 전 준비자료 → 국제상속 추가 진단으로 상품 계단을 명확히 한다.
- 전문가 디렉토리는 추천이 아니라 광고/제휴 목록으로만 운영한다.

## v0.6.6 리포트 샘플 PDF 체크

- 샘플 PDF는 법률자문, 세무자문, 유언장 작성 대행, 전문가 추천으로 오인되지 않게 고지한다.
- PDF는 '판정'이 아니라 '상담 전 확인할 신호와 준비자료 정리'로 표현한다.
- 유료 리포트의 상품성은 보여주되 실제 결제 전 과장 보장 문구를 넣지 않는다.
- 리포트 샘플은 쉬운 언어로 작성하고, 법률용어는 고객용 설명 뒤에 보조적으로만 둔다.
- 리포트 미리보기는 유료 전환 CTA를 강화하되, 결제·환불·품질 기준이 확정되기 전에는 결제를 붙이지 않는다.


## v0.6.8 결제 준비 체크

- 실제 PG 결제 전 상품 제공 범위, 환불정책, 개인정보 처리방침, 사업자 정보를 별도 고지한다.
- 결제 상품명은 법률자문, 세무자문, 유언장 작성 대행으로 오인되지 않게 한다.
- 결제 전 확인 체크박스로 정보성 리포트/상담 전 준비자료임을 명확히 한다.
- 전문가 목록은 추천/배정이 아니라 광고 또는 제휴 프로필이 포함된 디렉토리로 표시한다.
- 구매 의사 데이터는 상품별 수요 검증용으로 저장하고, 실제 결제 전에는 환불·제공범위·고지문을 재검토한다.


## v0.6.9 결제 전 고지/정책 체크

- PG 연동 전 상품 제공 범위, 환불 기준, 개인정보 처리, 사업자 표시를 분리해 고지한다.
- 상품 제공 범위는 정보성 리포트와 상담 전 준비자료로 제한한다.
- 법률자문, 세무자문, 유언장 작성 대행, 전문가 추천으로 오인될 문구를 제거한다.
- 디지털 리포트는 생성 전 취소 가능, 열람/다운로드 후 환불 제한 가능성, 사업자 귀책 사유 예외를 별도로 정리한다.
- 무료 진단 단계에서는 민감자료를 요구하지 않는다는 문구를 유지한다.
- 실제 결제 전 사업자명, 대표자, 사업자등록번호, 고객문의 연락처, 통신판매 관련 표시를 확정한다.
- 전문가 목록은 광고/제휴 프로필이 포함될 수 있음을 명확히 표시한다.


## v0.7.0 결제 전 동의·정책 체크

- 필수 동의: 상품 범위 확인, 환불 기준 확인, 개인정보 수집·이용 동의
- 선택 동의: 마케팅/리포트 출시 안내 수신
- 결제 전 상세 고지: 상품 제공 범위, 환불 기준, 개인정보, 사업자 표시 준비
- 금지 표현 유지: 법률자문, 세무자문, 유언장 작성 대행, 전문가 추천, 결과 보장
- 수익 전환 목적: 실제 PG 전 구매 의사·가격 저항·상품 선호 검증


## v0.7.1 결제 후 제공 화면 체크

- 결제 후 고객이 받는 화면에서 상품명, 제공 방식, PDF 다운로드, 이메일 안내, 환불 기준을 명확히 표시한다.
- 리포트는 정보성 자료이며 법률자문·세무자문·유언장 작성 대행·전문가 추천이 아님을 결제 후에도 재고지한다.
- 고객이 받은 자료 범위가 명확해야 환불·오해·법률자문 오인 리스크를 줄일 수 있다.


## v0.7.2 결제 오픈 대기 신청 검토
- 실제 결제 전 단계로 유지한다.
- 결제 오픈 알림 신청은 구매 확정이 아님을 명시한다.
- 관심 상품, 샘플 PDF 클릭, 대기 신청 의사를 answers.engagement에 저장한다.
- 법률자문·세무자문·유언장 작성 대행·전문가 추천 표현 금지를 유지한다.


## v0.7.3 관리자 통계 반영

- CSV를 열지 않아도 관리자 화면에서 전체 리드, 결제 오픈 대기 의사, 샘플 PDF 클릭, 국제상속 관련 신호를 확인한다.
- 상품별 선택 수와 샘플 PDF 클릭 수를 분리해 실제 결제 전 수요를 검증한다.
- 관리자 화면은 로컬 브라우저 백업 기준이며, 운영 배포 시 인증된 서버 관리자 화면으로 분리해야 한다.
- 전문가 목록 수익화는 광고/제휴 프로필 표시와 고객 직접 선택 원칙을 유지한다.
