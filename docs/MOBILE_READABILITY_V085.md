# v0.8.5 모바일 웹 가독성 추가 보강

PWA 기능은 추가하지 않았습니다.

## 반영 내용

- 첫 화면 핵심 안내 박스 추가
- 모바일 핵심 문구 대비 강화
- 질문/선택지/입력칸 글자 굵기와 터치 영역 강화
- 읽는 카드와 입력 영역의 역할 구분 강화
- 체크박스 문구 가독성 보강
- 법률자문/세무자문/유언장 작성 대행/전문가 추천 오인 표현 추가 없음

## 확인 기준

- 50~70대 사용자도 첫 화면 핵심 문구를 읽을 수 있는가
- 질문 선택지가 손가락으로 누르기 편한가
- 입력칸 위치와 라벨이 분명한가
- 안전 고지 문구가 너무 작지 않은가
