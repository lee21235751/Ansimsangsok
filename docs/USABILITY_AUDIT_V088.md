# v0.8.8 Usability and Public-Copy Audit

- PWA 기능은 추가하지 않음.
- 고객에게 미완성 서비스처럼 보일 수 있는 표현을 추가 정리함.
- 신청 전 동의 영역을 입력폼 가까이 이동해 모바일에서 찾기 쉽게 정리함.
- “구매 의사 저장”을 “알림 신청 저장”으로 변경함.
- “템플릿 기반” 등 내부 제작 방식 표현을 고객용 설명으로 완화함.
- 이름/연락처/이메일 placeholder와 hint를 더 직관적으로 조정함.
- 입력 영역, 필수 체크박스, 저장 버튼의 대비와 크기를 노안 기준으로 강화함.
