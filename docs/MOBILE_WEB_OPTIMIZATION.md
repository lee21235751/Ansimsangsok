# 안심상속 v0.8.3 모바일 웹 최적화

## 목적

PWA나 앱 설치 기능은 추가하지 않고, 모바일 브라우저에서 무료 진단 완료율과 결제 오픈 대기 신청 전환율을 높이는 것을 목표로 한다.

## 반영 내용

- 모바일 첫 화면 세로 길이 축소
- 모바일에서는 예시 카드 숨김 처리로 첫 CTA 노출 강화
- 질문 카드, 선택 버튼, 이전/다음 버튼 터치 영역 정리
- 하단 이전/다음 버튼을 모바일에서 sticky 처리
- 결과 점수 원형 크기 축소
- 리포트 미리보기, 상품 카드, 정책 카드, 제공 화면을 모바일 1열 구조로 정리
- 입력 폼과 체크박스 문구 가독성 개선
- 작은 화면에서 신뢰 배지와 부동산 세부 선택지가 과밀해지지 않도록 조정

## 사업자 리스크 검토

- 법률자문, 세무자문, 유언장 작성 대행 표현 추가 없음
- 전문가 추천/배정 표현 추가 없음
- 결제 기능 추가 없음
- 기존 개인정보 및 상품 범위 고지 유지

## 수익성 검토

- 모바일 진단 완료율 개선
- 샘플 PDF 클릭과 상품 관심 선택까지의 스크롤 피로 감소
- 알림 신청 폼 입력 편의 개선
- 실제 PWA 전환 전 모바일 웹 품질 우선 확보
