# 안심상속 검색/광고 공개 전 체크리스트

## 1. 사업자 리스크 기준

- 법률자문, 세무자문, 유언장 작성 대행으로 보이는 표현 금지
- 전문가 추천, 배정, 사건 연결 수수료로 보이는 표현 금지
- 유류분 가능/불가능, 유언장 유효/무효, 상속세 확정 계산 표현 금지
- 모든 유료 상품은 정보성 리포트, 상담 전 준비자료, 위험 신호 정리로 표현

## 2. 광고 문구 허용 방향

사용 가능:
- 최근 유류분 기준 변화, 우리 가족에게도 영향이 있을까요?
- 상속분쟁 위험 신호를 2분 만에 확인하세요.
- 부동산, 생전증여, 해외 거주 가족 요소를 쉽게 점검해보세요.
- 법률용어를 몰라도 이해할 수 있는 상속분쟁 사전 진단.

사용 금지:
- 상속분쟁 해결
- 유류분 청구를 막아드립니다
- 변호사 추천
- 공증 가능한 유언장 작성
- 상속세를 줄여드립니다
- 소송 걱정 끝
