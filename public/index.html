  • 포함: 진단 응답 요약, 유류분·부동산·국제상속 검토 신호, 준비자료 체크리스트, 전문가에게 물어볼 질문.
  • 제외: 법률자문, 세무자문, 유언장 작성 대행, 공증 대행, 전문가 추천·배정, 사건 결과 보장.

디지털 리포트는 생성·열람·다운로드 여부에 따라 환불 가능 범위가 달라질 수 있습니다.

  • 리포트 생성·열람 후: 디지털 콘텐츠 특성상 단순변심 환불 제한 가능성을 결제 전 고지합니다.
  • 중복결제, 미제공, 기술 오류 등 사업자 귀책 사유는 보정 또는 환불 기준을 별도로 둡니다.

개인정보·마케팅 동의 상세

무료 진단과 유료 안내 단계에서는 최소한의 정보만 수집하고, 마케팅 수신 동의는 선택으로 분리합니다.

  • 필수 수집: 이름, 연락처, 이메일, 진단 응답, 관심 상품, 저장 시각.
  • 미수집: 주민등록번호, 가족관계증명서, 등기부등본, 계좌정보, 정확한 주소.
  • 선택 동의: 리포트 출시, 결제 오픈, 서비스 안내 수신.

  • 전문가 목록에는 광고 또는 제휴 프로필 포함 가능성을 명확히 표시합니다.
  • 안심상속은 특정 전문가의 상담 결과, 선임 여부, 사건 결과를 보장하지 않는다는 고지를 유지합니다.

  • 리드 전체 조회는 브라우저 공개 코드에서 열지 않습니다.
  • 운영 관리자 화면은 서버 API 또는 Supabase Edge Function + 인증으로 분리합니다.
  • 전문가 목록/광고 상품 통계는 리드 개인정보와 분리해 관리합니다.
